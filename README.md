# LesQuiz
LesQuiz is a web-based online learning application that offers a lessons and quizzes about the four core subjects in school which is the English, Filipino, Mathematics, and Science to its users. Before taking the lessons, The user will need to create an account or to login an account and after that, the screen will shows the topics and lessons that needs to read by the user. Once the user is done reading, they will have a choice to take the quiz or to go back and choose another subject they preferred to read.
